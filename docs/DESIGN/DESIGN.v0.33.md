> ### M145 EquipmentSpec.note + PropertyPanel での表示
>
> 型 `EquipmentSpec` に optional `note: string` を追加。fetch 後の `byId` インデックスに
> 自動的に伝播。
>
> PropertyPanel (`FurnitureProperties`) で spec が見つかり、`spec.note` が空でなければ
> 「備考」行を追加表示:
>  - 文字色は `var(--gray-500)` (薄い)
>  - 文字サイズ 11px、line-height 1.4
>  - `whiteSpace: 'normal'` で複数行に折り返し
>  - alignItems: 'flex-start' で長文時のレイアウト崩れ防止
>
> 出典コード (例: `ソース側コード[A-空冷式パッケージ型空気]`) や旧 ID との関連
> (例: `A-101 と同一機器の可能性`) を点検作業時に確認できる。
>
> ### M146 BMS テンプレの ID 差し替え
>
> 既存テンプレ参照箇所:
>  - `bms-factory` / `bms-datacenter` / `bms-public` で `eq('E-105', ...)` → `eq('K-104', ...)`
>  - `PLACEMENT` マップに `K-104 / K-216 / A-109 / A-110 / A-111` (旧 ID 換算) と
>    `A-107 / A-108 / A-112 / A-113 / S-108 / K-019 / K-217` (新規) を追加
>
> ### v0.33 で扱わない範囲

>
